\section{引言：机器人的 Transformer 时代}

Ted Xiao 来自 Google Brain 机器人团队，介绍了 Transformer 和大型基础模型在机器人领域的应用。核心主题是：如何让机器人像语言模型一样，通过大规模数据和模型获得通用能力。

\begin{importantbox}{机器人学的独特挑战}
与纯数字领域（文本、图像）不同，机器人需要在\textbf{物理世界}中行动：
\begin{itemize}
    \item \textbf{实时性}：约 100ms 的推理延迟约束
    \item \textbf{安全性}：错误动作可能导致硬件损坏
    \item \textbf{数据稀缺}：无法像互联网文本一样大规模获取机器人操作数据
    \item \textbf{多模态}：需要融合视觉、语言、本体感觉、力反馈等
\end{itemize}
\end{importantbox}

\section{RT-1：Robotics Transformer}

\subsection{架构与训练}

\begin{importantbox}{RT-1 的设计}
RT-1（Robotics Transformer 1）是 Google 的大规模机器人操作模型：
\begin{itemize}
    \item 输入：RGB 图像 + 自然语言指令
    \item 输出：离散化的机器人动作（末端执行器位移、夹爪开合等）
    \item 训练数据：13 个机器人、17 个月的数据收集、超过 130K 演示
    \item 模型大小需控制在 100ms 推理时间内
\end{itemize}
\end{importantbox}

\subsection{缩放效应}

RT-1 展示了机器人领域的缩放效应：
\begin{itemize}
    \item 更多数据和更多样化的任务 $\rightarrow$ 更好的泛化
    \item 多任务训练优于单任务训练
    \item 但受限于实体数据收集的高成本
\end{itemize}

\subsection{本章小结}

RT-1 证明了 Transformer 可以用于真实世界的机器人控制，但数据获取仍是瓶颈。

\section{SayCan：语言模型作为机器人规划器}

\subsection{核心思想}

\begin{importantbox}{SayCan：接地的语言规划}
SayCan 将大语言模型用于机器人\textbf{任务规划}：
\begin{itemize}
    \item \textbf{Say}：LLM 评估每个可能动作的``有用性''（与任务目标的相关性）
    \item \textbf{Can}：价值函数评估每个动作的``可行性''（当前状态下能否成功执行）
    \item 最终选择 = $\arg\max(\text{有用性} \times \text{可行性})$
\end{itemize}
这将 LLM 的世界知识与机器人的物理能力``接地''（grounding）在一起。
\end{importantbox}

\subsection{本章小结}

SayCan 展示了 LLM 作为机器人高级规划器的潜力，通过``可行性过滤''解决了 LLM 不了解机器人能力的问题。

\section{Inner Monologue：闭环语言反馈}

\subsection{机器人的内心独白}

\begin{importantbox}{Inner Monologue 的创新}
在 SayCan 的基础上加入\textbf{闭环反馈}：
\begin{itemize}
    \item 机器人执行动作后，视觉检测器判断是否成功
    \item 成功/失败信息\textbf{反馈给 LLM}
    \item LLM 根据反馈重新规划后续动作
    \item 支持人类在执行中途改变指令
\end{itemize}
这使得机器人不再是``盲目执行预定计划''，而是能够\textbf{适应性地重规划}。
\end{importantbox}

\subsection{本章小结}

Inner Monologue 通过闭环反馈使机器人系统更加鲁棒和灵活。

\section{RT-2：视觉-语言-动作模型}

\subsection{从 VLM 到 VLA}

\begin{importantbox}{RT-2 的突破}
RT-2 将视觉-语言模型（VLM）直接微调为\textbf{视觉-语言-动作}（VLA）模型：
\begin{itemize}
    \item 基于 PaLI 或 PaLM-E 等 VLM
    \item 将机器人动作表示为文本 token（如``1 128 91 241 1 0 1''）
    \item 模型在预训练的互联网知识基础上，学习输出动作
    \item 展现出 zero-shot 泛化能力（如理解``移到 Taylor Swift 旁边''）
\end{itemize}
\end{importantbox}

\begin{warningbox}{上下文长度的限制}
当前 VLA 模型的一个瓶颈是\textbf{上下文长度}——仅 6 张图像就接近上下文容量上限，无法输入完整的操作轨迹。这限制了 few-shot 行为学习和长期规划。
\end{warningbox}

\subsection{本章小结}

RT-2 证明了互联网规模的视觉-语言预训练可以直接迁移到机器人控制，实现语义级别的泛化。

\section{总结与延伸}

Ted Xiao 的演讲展示了基础模型如何从三个层面改变机器人学：(1) RT-1 用 Transformer 做底层控制，(2) SayCan/Inner Monologue 用 LLM 做高层规划，(3) RT-2 将两者统一为端到端的 VLA 模型。机器人学正在经历从``为每个任务编程''到``通过数据和基础模型泛化''的范式转变。
