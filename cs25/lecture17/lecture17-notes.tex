\section{引言：AI 与医学的交汇}

Vivek Natarajan 来自 Google Health AI 团队。他分享了一个个人故事：在印度长大时父母希望他当医生，但他选择了计算机科学。最终，通过 AI + 医学的交叉研究，他找到了二者的结合。他预言 AI 最有可能首先在\textbf{医学领域}赢得诺贝尔奖。

\section{Med-PaLM：医学大语言模型}

\subsection{从通用 LLM 到医学 LLM}

\begin{importantbox}{Med-PaLM 的方法论}
Med-PaLM 基于 Google 的 PaLM/Flan-PaLM，通过以下方式适配医学领域：
\begin{itemize}
    \item \textbf{指令微调}：在医学问答数据上进行指令微调
    \item \textbf{Prompt Engineering}：精心设计的医学提示模板
    \item \textbf{专家评估}：由临床医生从多个维度评估模型输出
\end{itemize}
\end{importantbox}

\subsection{评估维度}

\begin{knowledgebox}{医学 AI 评估的特殊性}
医学 LLM 不能仅用准确率衡量，需要多维评估：
\begin{itemize}
    \item \textbf{科学共识一致性}：回答是否符合当前医学共识
    \item \textbf{知识检索能力}：是否准确引用了相关医学知识
    \item \textbf{推理质量}：推理过程是否合理
    \item \textbf{潜在危害}：回答是否可能导致患者伤害
    \item \textbf{偏见}：是否存在对特定人群的偏见
\end{itemize}
一个回答可以同时存在正确理解和错误推理的证据。
\end{knowledgebox}

\subsection{Med-PaLM 2 的改进}

\begin{importantbox}{Med-PaLM 2 的核心进步}
Med-PaLM 2 在多个评估维度上接近甚至达到临床专家水平：
\begin{itemize}
    \item 在美国医师执照考试（USMLE）上达到约 85\% 的准确率
    \item 在临床医生盲评中，Med-PaLM 2 的回答质量\textbf{显著缩小}与专家的差距
    \item 但在某些细微方面（如共情表达、个性化建议）仍有改进空间
\end{itemize}
\end{importantbox}

\subsection{本章小结}

Med-PaLM 系列证明了通用 LLM 可以通过适当的微调和提示工程适配到医学领域，但评估标准必须超越简单的准确率。

\section{蛋白质与基因组学}

\subsection{蛋白质语言模型}

\begin{knowledgebox}{蛋白质序列作为``语言''}
蛋白质可以看作由 20 种氨基酸组成的``句子''。Transformer 在蛋白质序列上的应用：
\begin{itemize}
    \item \textbf{ESM}（Meta）：在数十亿蛋白质序列上预训练的语言模型
    \item \textbf{AlphaFold}（DeepMind）：利用注意力机制预测蛋白质 3D 结构
    \item 关键洞察：蛋白质序列中的长程依赖（远距离氨基酸之间的相互作用）天然适合 Transformer
\end{itemize}
\end{knowledgebox}

\subsection{高效注意力机制}

Vivek 介绍了 \textbf{Performer} 中的线性注意力近似：

$$\text{Attention}(Q, K, V) \approx \phi(Q) \cdot (\phi(K)^T V)$$

其中 $\phi$ 是随机特征映射。这将注意力的复杂度从 $O(N^2)$ 降到 $O(N)$，使得处理超长蛋白质序列成为可能。

\subsection{本章小结}

Transformer 在生物信息学中的应用正在快速扩展，从蛋白质结构预测到基因组分析。

\section{医学影像与多模态}

\subsection{医学影像的 Transformer}

\begin{itemize}
    \item \textbf{病理切片分析}：超高分辨率全切片图像（WSI）的 Transformer 处理
    \item \textbf{放射学报告生成}：多模态模型将医学影像转化为临床报告
    \item \textbf{皮肤科诊断}：结合图像和病史的多模态推理
\end{itemize}

\subsection{本章小结}

医学影像是多模态 Transformer 的重要应用场景，高分辨率和专业领域知识是主要挑战。

\section{总结与延伸}

Vivek 的演讲展示了 Transformer 如何从多个角度改变医学 AI：从医学问答（Med-PaLM）到蛋白质科学（Performer、AlphaFold），再到医学影像分析。AI 在医学中的应用不仅需要模型能力，更需要严格的临床验证和伦理考量。
