\section{引言：从大语言模型到多模态}

Ming Ding（丁铭）是智谱 AI（Zhipu AI）的研究科学家，清华大学博士。他领导或参与了多个多模态模型研究，包括 CogView（文生图）、CogVideo（文生视频）、CogVLM（视觉语言模型）和 CogAgent（GUI 智能体）。本讲从 LLM 发展史出发，全面介绍多模态模型的技术路线。

\section{大语言模型发展简史}

\subsection{为什么 LLM 能成功}

\begin{importantbox}{LLM 成功的三大支柱}
\begin{enumerate}
\item \textbf{Transformer 架构}：并行计算 + 全局注意力
\item \textbf{规模化}（Scaling Laws）：更大的模型 + 更多数据 = 更好性能
\item \textbf{自回归预训练目标}：简单但通用的下一个 token 预测
\end{enumerate}
\end{importantbox}

\subsection{预训练实践技巧}

\begin{knowledgebox}{大模型训练的工程挑战}
训练百亿级参数模型需要：
\begin{itemize}
\item 数据并行 + 张量并行 + 流水线并行（3D 并行）
\item 混合精度训练（FP16/BF16 + FP32 主参数）
\item 梯度检查点（节省显存，以计算换空间）
\item 仔细的学习率调度和热身策略
\end{itemize}
\end{knowledgebox}

\subsection{本章小结}
LLM 的成功建立在简洁架构与工程规模化的结合之上。

\section{多模态理解：视觉-语言模型}

\subsection{主流架构范式}

当前视觉-语言模型（VLM）的主流架构：

\begin{enumerate}
\item 视觉编码器（如 ViT）提取图像特征
\item 投影层将视觉特征映射到语言模型的嵌入空间
\item 大语言模型处理文本和视觉 token 的混合序列
\end{enumerate}

\subsection{CogVLM}

\begin{importantbox}{CogVLM 的关键创新}
CogVLM 在 Transformer 的每一层为视觉 token 引入独立的 QKV 参数（视觉专家），而非简单地将视觉特征投影到语言嵌入空间。这种``深度融合''使视觉信息在每一层都被充分处理，而不仅仅在输入层。
\end{importantbox}

\subsection{GLM-4V 的简化架构}

\begin{knowledgebox}{高分辨率输入的处理}
智谱最新的 GLM-4V 采用了更简洁的架构——类似 LLaVA，但用步幅卷积（stride convolution）替代线性投影来处理高分辨率输入。这在保持简洁性的同时支持了更精细的视觉理解，特别在 OCR 相关任务上表现突出。
\end{knowledgebox}

\subsection{CogAgent：GUI 智能体}

CogAgent 是一个能操作计算机界面的视觉智能体：输入屏幕截图，输出操作指令（点击、输入文本等）。

\begin{itemize}
\item 输入：屏幕截图 + 自然语言任务描述
\item 输出：具体的界面操作序列
\item 应用：自动化网页操作、表单填写、信息检索
\end{itemize}

\subsection{本章小结}
VLM 架构正在从复杂走向简洁，深度融合和高分辨率处理是关键技术方向。

\section{多模态生成：文生图与文生视频}

\subsection{CogView 系列}

CogView 是中国最早的大规模文生图模型之一，基于自回归 Transformer 架构。随后的版本逐步转向扩散模型（Diffusion Model）。

\subsection{CogVideo}

\begin{importantbox}{文生视频的核心挑战}
视频生成比图像生成难得多：
\begin{itemize}
\item 时间一致性：帧间必须连贯
\item 计算成本：视频序列长度远大于单张图像
\item 物理真实性：物体运动应符合物理规律
\end{itemize}
CogVideo 通过时间注意力机制和帧间差分编码来保持时间一致性。
\end{importantbox}

\subsection{扩散模型 vs 自回归模型}

\begin{knowledgebox}{两种生成范式}
\textbf{自回归}：逐 token 生成，天然适合文本，但用于图像/视频时序列过长。\\
\textbf{扩散}：从噪声逐步去噪，适合连续信号（图像、视频），但不擅长离散符号。\\
当前趋势是融合两者：用自回归模型处理离散部分（文本指令），用扩散模型处理连续部分（图像/视频）。
\end{knowledgebox}

\subsection{本章小结}
多模态生成正在从纯文本扩展到图像和视频，扩散模型成为主要范式。

\section{物理世界理解}

\begin{warningbox}{当前多模态模型的局限}
现有的图像/视频理解模型本质上是在处理标注过的数据。对于未标注的海量视频数据（蕴含丰富的物理世界知识），我们缺乏有效的自监督学习方法来提取知识。这是通向真正的物理世界理解的关键瓶颈。
\end{warningbox}

\section{总结与延伸}

本讲全景展示了从 LLM 到多模态模型的技术演进路线。核心观点：(1) VLM 架构趋向简洁，深度融合是关键；(2) 视频生成的时间一致性仍是开放问题；(3) 从未标注视频中学习物理世界知识是下一个前沿。
