\section{引言：部分-整体层次结构问题}

Geoffrey Hinton 介绍了 \textbf{GLOM}——一个关于神经网络如何表示部分-整体层次结构的思想实验。这是一篇``设计文档''而非已实现的系统，但其中的思想深刻影响了后续研究。

\begin{importantbox}{核心问题}
人类视觉天然地将场景解析为层次结构：像素 $\rightarrow$ 边缘 $\rightarrow$ 部件（鼻子、眼睛）$\rightarrow$ 物体（脸）$\rightarrow$ 场景。如何让神经网络也表示这种\textbf{动态的、依赖于输入的}部分-整体关系？
\end{importantbox}

\begin{knowledgebox}{为什么这个问题难？}
传统 CNN 通过固定的层级结构处理视觉信息，但：
\begin{itemize}
    \item 同一个部件（如圆形）在不同上下文中可以是不同物体的一部分
    \item 层次结构应该是\textbf{动态的}——依赖于具体输入而非预先固定
    \item 经典的``胶囊网络''试图解决这个问题，但在实现上遇到困难
\end{itemize}
\end{knowledgebox}

\section{GLOM 的核心思想}

\subsection{岛屿与列}

GLOM 的架构可以想象为一个二维网格上的``列''，每个列对应输入的一个空间位置（类似 ViT 的 patch）。每个列有多个层级，每个层级用一个向量表示。

\begin{importantbox}{GLOM 的关键机制}
\begin{itemize}
    \item 每个空间位置有一个``列''，包含从低级到高级的多个表示层
    \item 同一层级上，相邻位置如果属于同一物体，其表示向量应该\textbf{趋于一致}——形成``岛屿''
    \item \textbf{同层注意力}：同一层级的相邻位置通过注意力机制互相靠拢
    \item \textbf{跨层预测}：高层预测低层（自上而下），低层预测高层（自下而上）
\end{itemize}
\end{importantbox}

\subsection{对比学习的角色}

Hinton 简要介绍了对比自监督学习的思想：
\begin{itemize}
    \item 同一图像的不同裁剪应有相似的表示
    \item 这种方法可以学习到有意义的特征，无需标签
    \item GLOM 中的``岛屿形成''类似于对比学习中的聚类过程
\end{itemize}

\subsection{本章小结}

GLOM 通过``岛屿''机制动态地形成部分-整体层次结构，而非依赖固定的网络拓扑。

\section{心理意象与变换随机场}

\subsection{心理意象（Mental Imagery）}

Hinton 讨论了神经网络如何表示\textbf{空间关系}：
\begin{itemize}
    \item 人类使用``心理意象''来推理空间关系（如家具的排列）
    \item 涉及关系推理的任务受益于某种内部的``画布''或``地图''
    \item GLOM 中的空间网格可以作为这种内部表示的载体
\end{itemize}

\subsection{变换随机场}

\begin{knowledgebox}{消歧义的挑战}
一个圆形可能是轮子的一部分、也可能是眼睛。Hinton 提出使用\textbf{变换随机场}来消歧义：节点之间的消息传递需要考虑空间变换。例如，``鼻子''向周围发送的``嘴巴应该在下方''的消息，需要根据空间位置关系进行变换。
\end{knowledgebox}

\subsection{神经场（Neural Fields）}

Hinton 将 GLOM 与神经场（NeRF 等）联系起来：
\begin{itemize}
    \item 高层表示可以通过给定空间位置来预测低层表示
    \item 这类似于 NeRF 中给定 3D 坐标预测颜色和密度
    \item 同一个高层向量可以在不同位置生成不同的低层预测
\end{itemize}

\subsection{本章小结}

GLOM 综合了对比学习、变换随机场和神经场等多个概念，构建了一个关于视觉层次表示的统一框架。

\section{总结与延伸}

GLOM 虽然是``纯粹的设计文档''，但提出了关于神经网络如何表示部分-整体关系的深刻思想。这些思想影响了后续关于视觉表示学习、对象中心学习和层次化表示的研究方向。
