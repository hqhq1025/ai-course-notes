\section{引言：常识推理与 ChatGPT 时代}

Yejin Choi 教授介绍了\textbf{神经符号常识推理}（Neurosymbolic Common Sense Reasoning）的研究。她首先回应了``ChatGPT 是否已解决 NLP''的疑问，通过一个经典的 Winograd Schema 例子展示：

\begin{warningbox}{ChatGPT 的常识脆弱性}
``The trophy doesn't fit in the brown suitcase because it's too big.''（it = trophy）

ChatGPT 在标准表述下回答正确，但稍微改变问法就会出错——说明它\textbf{没有真正理解物理常识}，而是在做模式匹配。这种``看似聪明实则脆弱''的行为是当前 LLM 的核心局限。
\end{warningbox}

\section{David vs. Goliath：小模型的逆袭}

\subsection{规模不是万能的}

\begin{importantbox}{规模 vs. 质量}
Yejin 认为``更大 = 更好''的叙事过于简单化：
\begin{itemize}
    \item 大模型在很多任务上表现优异，但在常识推理的\textbf{可靠性}上仍有严重问题
    \item 小模型通过更好的训练策略和数据质量，可以在特定任务上超越大模型
    \item 关键不在于参数数量，而在于模型\textbf{学到了什么样的知识}
\end{itemize}
\end{importantbox}

\subsection{本章小结}

在常识推理领域，模型规模不是唯一的解决方案——知识质量和推理机制同样重要。

\section{常识知识图谱与神经符号方法}

\subsection{ATOMIC 与 COMET}

\begin{importantbox}{构建常识知识}
\begin{itemize}
    \item \textbf{ATOMIC}：大规模常识知识图谱，包含日常事件的因果关系、意图、反应等
    \item \textbf{COMET}：在 ATOMIC 上训练的生成式常识模型——给定一个事件，可以推断相关的常识
    \item 核心思想：将\textbf{人类标注的常识}与\textbf{神经网络的泛化能力}结合
\end{itemize}
\end{importantbox}

\subsection{道德推理}

\begin{knowledgebox}{AI 的道德判断}
Yejin 介绍了将常识推理应用于\textbf{道德判断}的研究：
\begin{itemize}
    \item Delphi 系统：基于众包数据训练的道德判断模型
    \item 能处理自由形式的道德问题（如``杀熊是否道德？''vs.``为救孩子杀熊？''）
    \item 但仍然是\textbf{描述性的}（描述人们认为什么是对的）而非\textbf{规范性的}（什么真的是对的）
\end{itemize}
\end{knowledgebox}

\subsection{本章小结}

常识知识图谱和神经符号方法为补充 LLM 的常识短板提供了重要思路。

\section{价值多元主义与 AI 安全}

\subsection{谁的价值观？}

\begin{warningbox}{AI 伦理的复杂性}
在为 AI 注入道德判断时面临的根本问题：
\begin{itemize}
    \item 不同文化、宗教、个人有\textbf{不同的道德标准}
    \item 不应该将某一种道德框架强加为``正确的''
    \item 应采取\textbf{价值多元主义}——尊重多样化的文化和个人偏好
    \item 需要 AI、哲学、心理学、政策制定者的\textbf{跨学科合作}
\end{itemize}
\end{warningbox}

\subsection{AI 安全过滤器}

关于 AI 内容过滤：
\begin{itemize}
    \item 可以让 AI 过滤器\textbf{高度公平和包容}（避免微攻击等）
    \item 这\textbf{不违反}言论自由——只是让 AI 本身不说有害的话
    \item 但需要更多研究来确定``什么应该被过滤''
\end{itemize}

\subsection{本章小结}

AI 的道德推理需要在技术能力和价值多元性之间找到平衡。

\section{总结与延伸}

Yejin Choi 的演讲揭示了当前 LLM 在常识推理方面的深层局限，并展示了神经符号方法如何补充纯规模化方法的不足。常识推理、道德判断和价值对齐是 AI 发展中不可回避的核心挑战。
