\section{引言：开放科学推动 LLM 研究}

本讲由 UW/AI2 的 Hanna Hajishirzi 教授主讲，介绍开放训练配方如何改进语言模型的推理能力，涵盖预训练、后训练和推理时扩展三个阶段。

\begin{importantbox}{为什么需要完全开放的模型}
为促进创新和加速 LLM 科学研究，模型需要：\textbf{完全开放}（代码和数据）、\textbf{透明}、\textbf{可获取}和\textbf{可复现}。AI2 的 OLMo 和 Tulu 项目正是这一理念的实践。
\end{importantbox}

\section{后训练：Tulu 3 训练配方}

\subsection{三步训练流程}

\begin{importantbox}{Tulu 3 后训练管线}
\begin{enumerate}
    \item \textbf{指令微调}（SFT）：在多样化的指令-回答数据上微调
    \item \textbf{偏好调优}（DPO/PPO）：利用人类偏好数据对齐模型
    \item \textbf{可验证奖励强化学习}（RLVR）：利用可自动验证的任务（数学、代码）进行 RL 训练
\end{enumerate}
\end{importantbox}

\subsection{数据策略}

\begin{warningbox}{数据混合的挑战}
增加某类数据（如数学推理）可能导致其他能力（如创意写作）下降。需要精心平衡数据混合比例，这是一个活跃的研究方向。
\end{warningbox}

数据来源：人工标注 + 合成生成的混合方案最优。关键步骤包括：去污染（与评估集去重）、许可证检查、质量过滤。

\subsection{可验证奖励强化学习（RLVR）}

\begin{importantbox}{RLVR 的核心思想}
对于数学和编程等可自动验证答案的任务：
\begin{itemize}
    \item 模型生成解答
    \item 自动检查答案正确性作为奖励信号
    \item 使用 RL 算法（如 GRPO）优化
\end{itemize}
DeepSeek-R1 证明了这种简单方法的强大效果。
\end{importantbox}

\subsection{实验结果}

Tulu 3 应用于 Llama 3.1 405B 后，性能接近 GPT-4o，优于 DeepSeek-V3。

\section{预训练：OLMo 2}

OLMo 2 在 7B 和 13B 规模上，使用更少的训练 token，达到与 Llama 3 和 Qwen 2.5 相当的性能，位于帕累托最优曲线上。

\section{推理时扩展}

\begin{knowledgebox}{测试时计算扩展}
在推理阶段投入更多计算来提升性能，包括：
\begin{itemize}
    \item 多次采样 + 验证器选择（Best-of-N）
    \item 过程奖励模型（PRM）引导搜索
    \item 自我反思和迭代修正
\end{itemize}
\end{knowledgebox}

\section{总结与延伸}

\subsection{核心信息}

开放、可复现的训练配方对于推动 LLM 科学至关重要。预训练、后训练和推理时扩展三个阶段各有改进推理能力的方法，且大部分正交可叠加。
