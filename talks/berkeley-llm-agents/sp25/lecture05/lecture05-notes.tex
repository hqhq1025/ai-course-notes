\section{引言：从 AlphaGo 到 AlphaProof}

本讲由 Google DeepMind 的 Thomas Hubert 主讲，讲述从强化学习在游戏中的成功经验到将其应用于形式化数学证明的历程。

\begin{knowledgebox}{数学作为智能的根节点}
数学要求推理与规划、泛化与抽象、知识与创造力、开放且无界的复杂度，甚至需要一种美感。数学不仅是理解自然世界的根节点，也可能是通往智能的根节点。
\end{knowledgebox}

\section{数学形式化的历史弧线}

\subsection{从希腊人到计算机}

数学形式化经历了漫长的进化：
\begin{itemize}
    \item 公元前500年：希腊人发现\textbf{数学证明}的概念
    \item 欧几里得《几何原本》：公理化的起点
    \item 20世纪初：Hilbert 程序，现代公理化体系
    \item 符号化发展：从文字描述到代数符号（笛卡尔的变量记法）
    \item \textbf{计算机形式化}：数学形式化的下一个逻辑阶段
\end{itemize}

\subsection{Lean 与 Mathlib}

\begin{importantbox}{Lean 语言与 Mathlib 库}
\textbf{Lean} 是编程语言、定理证明器和交互式证明助手。\textbf{Mathlib} 是开源数学库，目标：
\begin{itemize}
    \item \textbf{通用性}：最弱假设、最强结论
    \item \textbf{统一性}：每个概念只有一个定义
    \item 已覆盖约80\%的本科数学课程
\end{itemize}
\end{importantbox}

\subsection{计算机形式化的变革潜力}

\begin{enumerate}
    \item \textbf{完美验证}：证明正确性由计算机保证
    \item \textbf{教育革命}：交互式学习，即时反馈
    \item \textbf{大规模协作}：Terence Tao 指出，形式化验证可使数学协作从5人扩展到数千人
    \item \textbf{将数学变成``电子游戏''}：关卡=定理，通关=证明，永恒有意义
\end{enumerate}

\section{强化学习的经验教训}

\subsection{AlphaZero 系列的核心哲学}

\begin{importantbox}{Zero 哲学}
如果一个 Agent 能从零开始掌握一个领域（无需任何人类信息），则证明该系统能够\textbf{自主发现和学习新知识}。这对于推进科学至关重要。
\end{importantbox}

AlphaZero 的成功案例：围棋、国际象棋、Atari、AlphaTensor（矩阵乘法算法发现）。

\subsection{使系统超越人类的四大要素}

\begin{enumerate}
    \item \textbf{大规模试错}：发现新知识需要尝试新事物并从反馈中学习
    \item \textbf{可靠的反馈/接地}：反馈必须是可靠的（非自我编造的）
    \item \textbf{搜索}：加速新模式和技能的发现
    \item \textbf{学习}：用搜索结果改进策略
\end{enumerate}

\begin{warningbox}{关键洞察：搜索与学习的飞轮}
搜索发现新的好路径 $\to$ 学习将这些路径内化为策略 $\to$ 更好的策略使搜索更高效 $\to$ 正反馈循环。这就是 AlphaZero 的核心机制。
\end{warningbox}

\section{AlphaProof：RL 遇上形式化数学}

\subsection{数学作为理想的 RL 环境}

Lean 提供了 RL 所需的一切：
\begin{itemize}
    \item \textbf{环境}：Lean 证明助手
    \item \textbf{动作}：证明策略（tactics）
    \item \textbf{反馈}：完美验证信号（证明通过/不通过）
    \item \textbf{无限复杂度}：从简单到 IMO 级别
\end{itemize}

\subsection{AlphaProof 的架构}

\begin{importantbox}{AlphaProof 系统}
\begin{enumerate}
    \item \textbf{形式化网络}：将非形式化数学问题翻译为 Lean 形式化声明
    \item \textbf{求解网络}：使用 RL 训练的模型生成形式化证明
    \item 关键洞察：即使形式化有误的问题也对训练有帮助
    \item \textbf{测试时 RL}：在实际竞赛中也可进行 RL 搜索
\end{enumerate}
\end{importantbox}

\subsection{IMO 2024 成绩}

AlphaProof + AlphaGeometry 达到\textbf{银牌水平}表现。

\section{总结与延伸}

\subsection{核心信息}

数学形式化和强化学习的结合正在开创全新的范式：完美验证提供可靠反馈，搜索与学习的飞轮驱动持续进步。这不仅能推动数学发现，还将重塑教育、协作和科学研究的方式。
