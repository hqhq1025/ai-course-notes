\section{引言：从语言模型到自主 Agent}

本讲由 CMU/Meta 的 Ruslan Salakhutdinov 教授主讲，介绍多模态自主 AI Agent，重点讨论 Web Agent 的评估、推理时搜索算法和数据扩展。

\begin{knowledgebox}{LLM Agent 的本质}
LLM Agent = 多轮 LLM + 工具使用。核心价值在于：
\begin{itemize}
    \item \textbf{动态计算时间}：思考 token 让模型对困难问题投入更多计算
    \item \textbf{外部工具信息}：运行代码、查看结果、修正方案
    \item \textbf{假设检验}：Agent 可提出假设、测试、根据结果修正
    \item \textbf{外部世界交互}：真正执行任务
\end{itemize}
\end{knowledgebox}

\section{VisualWebArena：多模态 Web Agent 评估}

\subsection{从 WebArena 到 VisualWebArena}

\begin{warningbox}{HTML 表示的问题}
HTML 源码杂乱（CSS、JavaScript 混杂），交互元素显示不正确，空间布局信息丢失，页面轻易超过 100K token。纯文本 Agent 难以有效处理。
\end{warningbox}

VisualWebArena 设计了需要\textbf{视觉理解}的任务：
\begin{itemize}
    \item 根据图片找到对应商品
    \item 阅读倒置的地址图片填写表单
    \item 理解图表数据跨网站检索信息
\end{itemize}

\subsection{环境设计}

环境建模为 POMDP（部分可观测马尔可夫决策过程），具有确定性转移函数。动作空间包括点击、悬停、输入文本、切换标签页、滚动等。

\section{推理时搜索：Tree Search for LLM Agents}

\begin{importantbox}{Agent 搜索的核心思想}
将 Web Agent 的决策建模为树搜索问题：
\begin{itemize}
    \item 每个节点是一个网页状态
    \item 每条边是一个动作
    \item 搜索目标是找到完成任务的动作序列
    \item 利用 LLM 作为策略网络和价值网络指导搜索
\end{itemize}
\end{importantbox}

由于 Web 环境的确定性，可以保存和恢复状态，使得树搜索实际可行。

\section{数据扩展：在线数据收集}

\begin{knowledgebox}{Agent 训练数据的瓶颈}
Agent 训练数据极度稀缺。解决方案包括：
\begin{itemize}
    \item 在模拟环境中自动收集交互轨迹
    \item 利用搜索算法生成高质量的成功轨迹
    \item 过滤和验证自动收集的数据
\end{itemize}
\end{knowledgebox}

\section{总结与延伸}

多模态 Web Agent 是 2025--2026 年的热点方向。评估需要真实的交互环境，推理时搜索可显著提升性能，数据扩展是训练高效 Agent 的关键瓶颈。
