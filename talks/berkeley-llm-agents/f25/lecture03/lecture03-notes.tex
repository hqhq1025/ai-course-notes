\section{引言}

本讲由 Google DeepMind 的 Oriol Vinyals 主讲，讨论 LLM 时代的多 Agent 系统。

\section{从 AlphaStar 到 LLM Agent}

\begin{knowledgebox}{AlphaStar 的经验}
在 StarCraft II 中，多 Agent 训练展示了涌现行为、策略多样性和自我对弈的价值。这些经验可迁移到 LLM Agent 领域。
\end{knowledgebox}

\section{LLM 多 Agent 框架}

\begin{importantbox}{多 Agent 协作模式}
\begin{itemize}
    \item \textbf{辩论}：多个 Agent 提出不同观点，通过辩论达成共识
    \item \textbf{分工}：不同 Agent 负责不同子任务
    \item \textbf{审查}：一个 Agent 生成，另一个 Agent 审查和改进
\end{itemize}
\end{importantbox}

\section{总结与延伸}

多 Agent 系统在 LLM 时代获得了新的生命力，涌现行为和集体智能是令人兴奋的研究方向。
