\section{引言}

本讲由 UT Austin 的 Peter Stone 教授主讲，从自主 Agent 的经典视角出发，讨论 Agent 的定义、架构和核心能力。

\section{自主 Agent 的定义}

\begin{importantbox}{Agent 的核心特征}
自主 Agent 是能够感知环境并采取行动以实现目标的系统。关键属性：自主性、反应性、主动性、社交能力。
\end{importantbox}

\section{Agent 架构}

从反射式到 BDI（信念-愿望-意图）到基于学习的架构，LLM Agent 代表了最新的架构范式。

\section{多 Agent 系统}

协作、竞争和混合场景下的多 Agent 交互，包括通信协议、协调机制和博弈论视角。

\section{总结与延伸}

LLM Agent 继承了经典 Agent 研究的许多核心问题（规划、学习、协调），同时引入了新的能力和挑战。
