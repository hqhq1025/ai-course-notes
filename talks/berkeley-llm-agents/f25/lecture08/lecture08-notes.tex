\section{引言}

本讲由微软的 Weizhu Chen 主讲，讨论如何训练具有 Agent 能力的模型。

\section{Agent 能力的训练方法}

\begin{importantbox}{训练 Agentic 模型的关键要素}
\begin{itemize}
    \item \textbf{工具使用训练}：教模型调用 API、执行代码、浏览网页
    \item \textbf{多轮交互训练}：模拟 Agent 与环境的多步交互
    \item \textbf{规划能力训练}：分解任务和制定执行计划
    \item \textbf{错误恢复训练}：从失败中学习和修正
\end{itemize}
\end{importantbox}

\section{数据策略}

\begin{knowledgebox}{Agent 训练数据来源}
\begin{itemize}
    \item 人工标注的 Agent 轨迹
    \item 合成轨迹生成
    \item 环境交互中的自我学习
    \item 弱监督和远程监督
\end{itemize}
\end{knowledgebox}

\section{总结与延伸}

训练 Agentic 模型需要超越传统的指令微调，引入多轮交互、工具使用和错误恢复的训练范式。
