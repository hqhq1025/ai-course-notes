\section{引言}

本讲由 Stanford 的 James Zou 教授主讲，讨论如何用 AI Agent 自动化科学研究。

\section{科学发现的自动化}

\begin{importantbox}{AI Agent 在科学中的角色}
\begin{itemize}
    \item \textbf{假设生成}：基于文献和数据提出新假设
    \item \textbf{实验设计}：优化实验参数和流程
    \item \textbf{数据分析}：自动化统计分析和模式发现
    \item \textbf{文献综述}：大规模文献检索和综合
\end{itemize}
\end{importantbox}

\section{案例研究}

AI Agent 在生物学、化学和材料科学中的应用案例，包括药物发现和蛋白质设计。

\begin{warningbox}{科学 Agent 的验证挑战}
科学假设的验证需要实验，而实验成本高、周期长。如何在有限实验预算下最大化发现效率是核心挑战。
\end{warningbox}

\section{总结与延伸}

AI Agent 有潜力加速科学发现的每个阶段，但需要可靠的验证机制和人类专家的持续参与。
