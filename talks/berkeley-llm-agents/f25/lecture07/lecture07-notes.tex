\section{引言}

本讲由 OpenAI 的 Noam Brown 主讲（Libratus/Pluribus 扑克 AI 作者），讨论多 Agent AI 的前沿进展。

\section{从博弈论到多 Agent AI}

\begin{knowledgebox}{不完美信息博弈的经验}
德州扑克 AI（Libratus/Pluribus）展示了在不完美信息环境中通过搜索和自我对弈实现超人表现的可能性。
\end{knowledgebox}

\section{LLM 中的多 Agent 方法}

\begin{importantbox}{搜索与推理}
\begin{itemize}
    \item 推理时搜索（Test-Time Search）可大幅提升 LLM 性能
    \item 自我对弈和辩论可提升推理质量
    \item 多 Agent 审查和改进机制
\end{itemize}
\end{importantbox}

\section{总结与延伸}

博弈论和搜索方法的洞察正在深刻影响 LLM Agent 的设计。推理时计算和多 Agent 交互是提升 AI 能力的关键方向。
