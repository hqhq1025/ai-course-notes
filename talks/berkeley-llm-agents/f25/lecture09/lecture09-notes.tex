\section{引言}

本讲由 UC Berkeley 的 Jiantao Jiao 教授主讲，讨论如何通过后训练方法构建可验证的 Agent。

\section{可验证性的重要性}

\begin{importantbox}{为什么 Agent 需要可验证}
Agent 采取真实行动，错误可能造成不可逆后果。可验证性提供了一种确保 Agent 行为正确的机制。
\end{importantbox}

\section{后训练中的验证方法}

\begin{itemize}
    \item \textbf{形式化验证}：数学证明 Agent 行为满足特定属性
    \item \textbf{运行时验证}：在执行过程中检查约束条件
    \item \textbf{基于奖励的验证}：利用可验证的奖励信号进行 RL 训练
\end{itemize}

\begin{warningbox}{验证的可扩展性挑战}
形式化验证在简单系统上可行，但对复杂的 LLM Agent 系统，完整的形式化验证仍然极具挑战。需要在严格性和可扩展性之间找到平衡。
\end{warningbox}

\section{总结与延伸}

可验证的 Agent 是实现可信 AI 的关键。后训练方法通过将验证信号融入训练过程，提供了一条务实的路径。
