\section{引言}

本讲由 Sida Wang 主讲，深入分析 LLM 基准测试中的可预测噪声问题。

\section{基准测试的噪声来源}

\begin{importantbox}{LLM 基准中的噪声}
\begin{itemize}
    \item \textbf{数据污染}：测试数据出现在训练集中
    \item \textbf{评估方差}：采样、Prompt 格式、温度参数带来的随机性
    \item \textbf{过拟合}：针对特定基准优化而非真实能力提升
    \item \textbf{评分标准模糊}：开放式任务的评估主观性
\end{itemize}
\end{importantbox}

\section{改进评估的方法}

\begin{knowledgebox}{更可靠的评估策略}
\begin{itemize}
    \item 持续更新基准以防泄露
    \item 统计严格的置信区间报告
    \item 多维度评估而非单一分数
    \item 人类评估作为锚点
\end{itemize}
\end{knowledgebox}

\section{总结与延伸}

理解和量化基准测试中的噪声对于正确解读 LLM 进展至关重要。社区需要更严格的评估方法论。
