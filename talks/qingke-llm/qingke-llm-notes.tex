\section{引言}
本场是青稞 AI 嘉年华的 LLM/MLLM 专题圆桌讨论，由陈嘉乐主持，邀请多位嘉宾探讨大模型和多模态模型的最新进展。讨论围绕三大主题：训练方式、后训练突破、以及 Agentic \& Tool Use。

\section{2025 年的技术图景}

\begin{importantbox}{2025 年：按了加速键}
从 DeepSeek R1 的突破开始，整个行业加速发展。开源模型在 Reasoning 领域迅速追上了 o1/o3 等闭源模型。文生图、文生视频等应用也蓬勃发展。``AI 一年，人间三年''。
\end{importantbox}

\subsection{各嘉宾视角}
\begin{itemize}
  \item \textbf{曹玉}：强化学习和后训练方向，2025 年是开花结果的一年
  \item 开源模型的发展速率远未放缓
  \item 头部模型的能力边界仍在快速扩展
\end{itemize}

\section{训练方式的演进}

\subsection{预训练的变化}
\begin{knowledgebox}{预训练趋势}
\begin{itemize}
  \item 数据质量越来越重要（合成数据、数据清洗）
  \item Scaling Law 仍在发挥作用
  \item MoE 架构成为主流
\end{itemize}
\end{knowledgebox}

\subsection{本章小结}
预训练从``堆数据''转向``精细化数据工程''，模型架构也在持续演进。

\section{后训练的突破}

\subsection{RL Post-Training}
\begin{importantbox}{RL 后训练的范式变化}
\begin{itemize}
  \item DeepSeek R1 开创了 RL 后训练的新范式
  \item GRPO 等算法简化了训练流程
  \item Reward Model 的设计越来越重要
  \item 多阶段训练（SFT $\to$ RL）成为标准流程
\end{itemize}
\end{importantbox}

\subsection{Reasoning 模型}
Thinking Model 成为 2025 年的主旋律：从 o1 到 R1，从 Long CoT 到 Interleaved Thinking。

\subsection{本章小结}
后训练是 2025 年进展最快的方向，RL + Reasoning 是核心。

\section{Agentic \& Tool Use}

\subsection{Agent 的发展}
从概念验证到实际落地，Agent 在 2025 年取得了实质性进展。

\subsection{多模态进展}
MLLM（多模态大语言模型）在图像理解、视频理解等方面持续突破。

\subsection{本章小结}
Agentic 和多模态是 LLM 应用落地的两大方向。

\section{总结与延伸}
\begin{enumerate}
  \item 2025 年是大模型加速发展的一年
  \item RL 后训练和 Reasoning 模型是核心突破
  \item Agentic 和多模态是应用落地的关键方向
  \item 开源模型生态蓬勃发展
\end{enumerate}
