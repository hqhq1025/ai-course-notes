\section{引言}
本场是青稞 AI 嘉年华的 Infra 专题，聚焦大模型推理和训练基础设施的最新进展。

\section{嘉宾与研究方向}

\subsection{张明星（清华大学副教授）}
\begin{knowledgebox}{MoonCake \& KTransformers}
张明星团队发起了两个重要的开源项目：
\begin{itemize}
  \item \textbf{MoonCake}：云端分布式集群推理优化
  \item \textbf{KTransformers}：边缘端 CPU-GPU 异构推理优化
\end{itemize}
主要聚焦推理场景，2025 下半年也开始做 RL Infra 和 Agent Infra。
\end{knowledgebox}

\subsection{其他嘉宾}
来自学术界和工业界的 Infra 专家分享各自方向的最新进展。

\section{2025 年 Infra 领域关键进展}

\subsection{PD 分离（Prefill-Decode Disaggregation）}
\begin{importantbox}{PD 分离}
将 Prefill 和 Decode 两个阶段分离到不同的硬件/集群上执行。Prefill 阶段计算密集，适合高算力节点；Decode 阶段内存带宽密集，适合高带宽节点。PD 分离在 2025 年已成为 well-known 的技术。
\end{importantbox}

\subsection{大 EP（Expert Parallelism）}
MoE 模型的 Expert 并行：将不同 Expert 分配到不同 GPU 上，实现高效的 MoE 推理。

\subsection{RL Infra}
\begin{warningbox}{RL Infra 的核心瓶颈}
现在 RL 训练的核心卡点在 rollout（推理）上，而不是训练本身。高效的推理引擎（如 vLLM、SGLang）是做好 RL Training 的前提。
\end{warningbox}

\subsection{Agent Infra}
Agent 场景对 Infra 提出新要求：多轮对话、工具调用、异步执行、状态管理等。

\subsection{本章小结}
PD 分离、大 EP、RL Infra 是 2025 年 Infra 领域的三大关键技术方向。

\section{云端 vs. 边缘端推理}

\subsection{云端推理}
大规模集群部署，追求高吞吐量。PD 分离、大 EP 是核心优化手段。

\subsection{边缘端推理}
本地设备部署，追求低延迟和隐私保护。CPU-GPU 异构计算是关键技术。

\subsection{本章小结}
云端和边缘端有不同的优化目标和技术路线，但推理优化是共同的核心主题。

\section{总结与延伸}
\begin{enumerate}
  \item PD 分离和大 EP 已成为推理优化的标准技术
  \item RL Infra 的核心瓶颈在推理（rollout）
  \item Agent 场景对 Infra 提出了新的需求
  \item 云端追求吞吐，边缘追求延迟，各有优化路线
\end{enumerate}
