\section{引言}
本期介绍 veRL 训练启动脚本中琳琅满目的参数，深入理解每个参数的原理层含义。

\begin{importantbox}{参数理解的重要性}
很多同学已经训练了很多模型，但对参数的原理层含义理解很浅。理解好 veRL 的参数后，理解 TRL 等其他框架也会非常清晰。
\end{importantbox}

\section{官方文档资源}
veRL 提供了几个关键文档：
\begin{itemize}
  \item Config 文档：\texttt{ppotrainer.yaml} 所有参数配置及含义
  \item Algorithm 文档：各种 Advantage Estimator 的说明
\end{itemize}

\section{关键训练参数}
\subsection{模型相关参数}
包括 model path、tokenizer、dtype 等基础配置。

\subsection{算法相关参数}
PPO/GRPO 的 clip epsilon、KL coefficient、learning rate 等。

\subsection{训练相关参数}
Batch size、gradient accumulation、number of rollouts 等。

\subsection{本章小结}
参数分为模型、算法、训练三大类。深入理解每个参数有助于调优训练效果。

\section{总结与延伸}
\begin{enumerate}
  \item veRL 参数分为 Config、Algorithm、Training 三类
  \item 理解参数原理是调优的基础
  \item 框架之间原理相通，veRL 经验可迁移到 TRL
\end{enumerate}
