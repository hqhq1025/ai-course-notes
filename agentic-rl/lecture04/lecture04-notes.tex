\section{引言}
本期将 REINFORCE 算法应用到语言模型中，实现一个简单效果：不管输入什么 prompt，模型输出都包含特定语义（如 ``mode）。

\section{Reward 环境设计}
\subsection{基于 Cosine Similarity 的 Reward}
\begin{importantbox}{Reward 定义}
\begin{enumerate}
  \item 语言模型 rollout 生成一个序列（句子）
  \item 计算该句子的 embedding
  \item 计算目标词（如 ``mode）的 embedding
  \item 两者的 cosine similarity 作为 reward
\end{enumerate}
\end{importantbox}

\section{REINFORCE 实现}
\subsection{算法流程}
\begin{enumerate}
  \item 从当前策略 $\pi_\theta$ 采样生成序列
  \item 计算每个序列的 reward（cosine similarity）
  \item 计算 PG Loss 并反向传播更新 $\theta$
\end{enumerate}

\subsection{本章小结}
REINFORCE 是最朴素的 PG 算法，可以直接应用于语言模型，将 RL 的概念落地到实际的文本生成任务中。

\section{总结与延伸}
\begin{enumerate}
  \item REINFORCE 可以直接用于语言模型的 RL 训练
  \item Reward 环境可以用 embedding similarity 定义
  \item 这是理解更复杂的 PPO/GRPO 的基础
\end{enumerate}
