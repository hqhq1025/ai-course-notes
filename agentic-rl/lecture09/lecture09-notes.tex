\section{引言}
本期通过 Bradley-Terry 模型重温 Reward Model，梳理概率统计建模、深度学习和最大似然估计之间的关系。

\begin{importantbox}{三者之间的关系}
\begin{enumerate}
  \item \textbf{概率统计建模}：定义数据的生成过程（如 Bradley-Terry 偏好模型）
  \item \textbf{最大似然估计 (MLE)}：基于统计模型定义 loss function
  \item \textbf{深度学习}：作为求解工具，通过梯度下降优化 loss
\end{enumerate}
\end{importantbox}

\section{Bradley-Terry 偏好模型}
给定两个 response $y_w$（preferred）和 $y_l$（rejected），Bradley-Terry 模型定义偏好概率：
$$P(y_w \succ y_l) = \sigma(r(y_w) - r(y_l))$$
其中 $r(\cdot)$ 是 Reward Model 的输出，$\sigma$ 是 sigmoid 函数。

\section{从 MLE 到 Loss}
最大化似然 $\Leftrightarrow$ 最小化负对数似然：
$$\mathcal{L} = -\log \sigma(r(y_w) - r(y_l))$$

\subsection{本章小结}
Reward Model 的训练本质上是概率统计建模 + MLE 求解。深度学习只是优化工具。

\section{总结与延伸}
\begin{enumerate}
  \item 掌握概率统计建模是做好 RL 研究的基础
  \item Bradley-Terry 是 Reward Model 的标准建模方法
  \item 深度学习是求解工具，概率模型才是核心
\end{enumerate}
