\section{引言}
本期从概率分布的视角重新审视语言模型的 SFT 和 RL 训练过程。

\begin{importantbox}{分布视角}
无论是 SFT 还是 RL，都是将 Base Model 的分布 $\pi_{\text{ref}}$ 训练到一个新的目标分布：
\begin{itemize}
  \item \textbf{SFT}：模仿人类数据分布的``平均形态（监督学习）
  \item \textbf{RL}：搜索奖励最高的分布（强化学习）
\end{itemize}
两者意味着不同的优化过程和不同的目标分布。
\end{importantbox}

\section{SFT 的分布含义}
SFT 通过 Prompt-Response 数据对模型做监督学习。目标是让模型输出分布尽可能接近训练数据的分布。

\section{RL 的分布含义}
RL 中 Response 是模型 online rollout 出来的（不是固定数据）。目标是最大化期望奖励，最终分布不一定等于人类分布。

\subsection{本章小结}
SFT 追求``像人类，RL 追求``奖励最高。两者的目标分布本质不同。

\section{总结与延伸}
\begin{enumerate}
  \item 分布视角是理解 SFT vs. RL 的统一框架
  \item SFT = 监督模仿；RL = 奖励最大化
  \item 为理解 DPO 做铺垫
\end{enumerate}
