## Core Teaching Axes

- Agent definition: perception + action; observation → decision → action loop.
- Agent types: coding, search, tool-use, computer-use.
- Training routes: in-context learning vs end-to-end training.
- Training ingredients: synthetic trajectories, reinforcement learning, safety/sandbox/human-in-loop.
- New paradigm: environment + task/reward construction, self-improvement, rubrics as reward, context engineering.
