\section{从 Actor-Critic 到纯 Value-Based 方法}

前几讲介绍了 actor-critic 方法：同时维护策略（actor）和价值函数（critic）。Q-Learning 走得更远——\textbf{完全去掉显式策略}，只学习 Q 函数，然后从 Q 函数隐式地派生策略。

\begin{knowledgebox}{Value-Based vs. Actor-Critic}
\begin{itemize}
    \item \textbf{Actor-Critic}：同时学习策略 $\pi_\theta$ 和价值函数 $\hat{V}$ 或 $\hat{Q}$。
    \item \textbf{Value-Based (Q-Learning)}：只学习最优 Q 函数 $Q^*$，策略隐式为 $\pi(s) = \arg\max_a Q^*(s, a)$。
\end{itemize}
\end{knowledgebox}

\subsection{本章小结}

Q-Learning 是一种纯 value-based 方法，不需要显式的策略网络。

\section{Q-Learning 基础}

\subsection{最优 Q 函数}

\begin{importantbox}{Bellman 最优方程}
最优 Q 函数 $Q^*$ 满足：
$$Q^*(s, a) = r(s, a) + \gamma \max_{a'} Q^*(s', a')$$

关键性质：$Q^*$ 对应的策略 $\pi^*(s) = \arg\max_a Q^*(s, a)$ 是所有策略中最优的。
\end{importantbox}

与 actor-critic 中拟合 $Q^\pi$（当前策略的 Q 函数）不同，Q-Learning 直接拟合 $Q^*$（最优策略的 Q 函数），使用 max 而不是 $\mathbb{E}_{a' \sim \pi}$。

\subsection{DQN 算法}

Deep Q-Network (DQN) 使用深度神经网络来参数化 $Q_\phi(s, a)$：

$$\min_\phi \sum_{(s,a,r,s') \in \mathcal{B}} \left\| Q_\phi(s, a) - \left(r + \gamma \max_{a'} Q_{\phi'}(s', a')\right) \right\|^2$$

\begin{importantbox}{DQN 的关键技巧}
\begin{itemize}
    \item \textbf{Replay Buffer}：存储所有历史转移，随机采样来打破数据相关性。
    \item \textbf{Target Network}：使用延迟更新的 $Q_{\phi'}$ 计算 TD target，避免"自己追自己"的不稳定。
    \item \textbf{$\epsilon$-Greedy 探索}：以概率 $\epsilon$ 随机选择动作，否则选 $\arg\max_a Q(s,a)$。
\end{itemize}
\end{importantbox}

\subsection{本章小结}

Q-Learning 通过拟合最优 Q 函数来间接获得最优策略。DQN 将 Q-Learning 扩展到高维状态空间，replay buffer 和 target network 是关键的稳定化技巧。

\section{连续动作空间的挑战}

DQN 原始形式只适用于\textbf{离散}动作空间（网络输出每个动作的 Q 值，然后取 argmax）。对于连续动作空间，$\max_a Q(s,a)$ 本身就是一个优化问题。

解决方案包括：
\begin{enumerate}
    \item \textbf{离散化}：将连续空间离散化，但维度诅咒严重。
    \item \textbf{采样}：随机采样多个动作，取 Q 值最大的。
    \item \textbf{学习一个 actor}：回到 actor-critic 框架（如 SAC）。
    \item \textbf{DDPG}：训练一个确定性策略网络 $\mu_\theta(s) = \arg\max_a Q(s,a)$。
\end{enumerate}

\begin{warningbox}{Q-Learning 在连续空间的困难}
纯 Q-Learning 在连续动作空间中需要解决内层优化问题 $\max_a Q(s,a)$，这在高维空间中计算代价很大。这也是为什么连续控制任务中 actor-critic 方法（如 SAC）通常比纯 Q-learning 更受欢迎。
\end{warningbox}

\subsection{本章小结}

Q-Learning 在离散动作空间中自然高效，但在连续动作空间中需要额外的技巧或回归到 actor-critic 方法。

\section{Double Q-Learning 与过估计问题}

\subsection{过估计偏差}

$$\max_{a'} Q_\phi(s', a') \geq Q^*(s', \pi^*(s'))$$

取 max 操作会系统性地高估 Q 值，因为它选择的是\textbf{噪声最大的方向}。

\subsection{Double DQN}

使用当前网络选择动作，但用 target 网络评估：
$$y = r + \gamma Q_{\phi'}\left(s', \arg\max_{a'} Q_\phi(s', a')\right)$$

这在一定程度上缓解了过估计问题。

\subsection{本章小结}

Q-Learning 有系统性的过估计偏差。Double Q-Learning 通过分离动作选择和动作评估来缓解这一问题。

\section{总结与延伸}

\begin{enumerate}
    \item Q-Learning 是一种纯 value-based 方法，直接学习最优 Q 函数。
    \item DQN 使用 replay buffer + target network + $\epsilon$-greedy 探索，是深度 Q-Learning 的里程碑。
    \item Q-Learning 天然适合离散动作空间；连续空间需要额外处理。
    \item 过估计是 Q-Learning 的固有问题，Double DQN 等方法可以缓解。
    \item Q-Learning 是完全 off-policy 的，数据效率高。
\end{enumerate}
